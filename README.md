# Sistema de Delivery
O Sistema de Delivery em Java é uma aplicação de console desenvolvida como projeto de estudo para simular o funcionamento de um serviço de delivery de restaurantes. Nele é possível registrar restaurantes, clientes e pratos, criar pedidos associados a esses cadastros e acompanhar todo o ciclo de vida do pedido, desde a criação até a entrega. Cada pedido guarda informações como prato solicitado, cliente, restaurante, status atual, tempo estimado de entrega, data do pedido e, quando concluído, a data de entrega efetiva.
O sistema possui um gerenciador de entregas responsável por organizar os pedidos em listas de “em andamento”, “entregues” e “atrasados”, com base na comparação entre a data do pedido, o tempo estimado e o horário atual. A partir desse controle, é possível identificar pedidos atrasados, atualizar o status para entregue, mover pedidos entre as listas de monitoramento e gerar representações textuais (toString) com os principais dados do pedido para fins de consulta e relatório.
